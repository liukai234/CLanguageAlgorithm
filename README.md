## 项目说明
题目五：家族关系查询系统

1.【问题描述】
建立家族关系数据库，实现对家族成员关系的相关查询。

2.【基本要求】
建立家族关系并存储到文件中；实现家族成员的添加；查询家族成员的双亲、祖先、兄弟、孩子及后代信息。
